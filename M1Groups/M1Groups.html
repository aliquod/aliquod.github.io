Group Axioms

Binary operations

  • A binary operation ∗*∗﻿ on a set SSS﻿ is a map S×S→SS\times S\to SS×S→S﻿, and ∗(a,b)*(a,b)∗(a,b)﻿ is usually written as a∗ba*ba∗b﻿
  • A binary operation ∗*∗﻿ can be associative and/or commutative (aka Abelian)
  • a binary operation can have identity elements, and elements that are inverses of others
    • inverses under associative operations are unique
  • A subset TTT﻿ of SSS﻿ is closed under ∗*∗﻿ if restriction of ∗*∗﻿ to TTT﻿, ∗T*_T∗T​﻿, is a binary operation too (i.e. ∗T:T×T→T*_T:T\times T\to T∗T​:T×T→T﻿)

Group axioms

  • A group is a set GGG﻿ paired with a binary operation ∗*∗﻿ on it, with axioms:
    • ∗*∗﻿ is associative, has identity e∈Ge\in Ge∈G﻿
    • all elements in GGG﻿ has inverses
    • (and GGG﻿ closed under ∗*∗﻿, implied by ∗*∗﻿ being a binary operation over GGG﻿)

Properties

  • A group GGG﻿ is cyclic if there exists g∈Gg ∈ Gg∈G﻿ such that G={gk:k∈Z}G = \{g^k : k ∈ \Z\}G={gk:k∈Z}﻿, where ggg﻿ is called the generator
    • e.g. Z\ZZ﻿ is cyclic with generators 1,−11,-11,−1﻿ (either works)
  • the Cayley table (or group table) of GGG﻿ is a square grid which contains all the possible products of two elements from GGG﻿
    • a Cayley group table is a latin square: every group element appears precisely once in each row and in each column
  • The order of an element g∈Gg\in Gg∈G﻿, o(g)o(g)o(g)﻿, is the least positive integer rrr﻿ such that gr=eg^r = egr=e﻿. If no such integer exists then ggg﻿ has infinite order

Cycles

  • A permutation in SnS_nSn​﻿ is a cycle if it cycles some subset of {1,…,n}\{1,\dots,n\}{1,…,n}﻿, while keeping the remaining unchanged
    • a kkk﻿-cycle cycles kkk﻿ elements
  • for cycles, the cycle notation lists the elements it cycles, while the unchanged elements are omitted
  • two cycles are disjoint if they operate on completely different elements
    • disjoint cycles commute (see lecture notes for proof)

Decomposing permutations as cycles

  • all permutations can be uniquely written as the product of disjoint cycles (up to commuting)
    • proof: for an element aaa﻿, the set {a,aσ,aσ2,… }\{a,a\sigma,a\sigma^2,\dots\}{a,aσ,aσ2,…}﻿ will eventually have repetitions, since the original set {1,…,n}\{1,\dots,n\}{1,…,n}﻿ is finite.

      take k+1k+1k+1﻿ as the first time a repeat occurs, then (a,aσ,…,aσk)(a,a\sigma,\dots,a\sigma^k)(a,aσ,…,aσk)﻿ is a part of the permutation

      now take bbb﻿ not attained in (a,aσ,…,aσk)(a,a\sigma,\dots,a\sigma^k)(a,aσ,…,aσk)﻿, repeat to get (b,bσ,…,bσl)(b,b\sigma,\dots,b\sigma^l)(b,bσ,…,bσl)﻿, which is disjoint from the cycle of aaa﻿ for otherwise bbb﻿ would be attained somewhere in it

      repeat to get more disjoint cycles till {1,…,n}\{1,\dots,n\}{1,…,n}﻿ is exhausted

      since the cycle in which one element appears is unique, the entire expression of the permutation as cycles is unique

    • in writing the permutation as disjoint cycles, unchanged elements (1-cycles) are omitted
  • the cycle decomposition type (or just cycle type) of a permutation is the number of cycles of each length in its decomposition as disjoint cycles (not the cycles themselves)
  • the order (of a permutation) is the least common multiple of the cycle lengths of its cycle types
  • two permutations σ,τ∈Sn\sigma,\tau\in S_nσ,τ∈Sn​﻿ are conjugates if there is ρ∈Sn:σ=ρ−1τρ\rho\in S_n:\sigma=\rho^{-1}\tau\rhoρ∈Sn​:σ=ρ−1τρ﻿
    • conjugate permutations have the same cycle type

Parity of permutations

  • A transposition is another term for a 2-cycle
    • a transposition has permutation matrix of determinant -1
  • Every permutation can be written as a product of transpositions
    • proof: write the permutation as disjoint cycles, then write the cycles as consecutive transpositions
  • A permutation is odd (even) if it can be written as a product of an odd (even) number of transpositions
    • The parity (even-or-odd-ness) of a permutation is well-defined, since the cycle decomposition is unique, and each cycle of length kkk﻿ always decomposes into k−1k-1k−1﻿ transpositions, so the total number is determined
    • Alternatively, the det⁡\detdet﻿ of the permutation matrix of the permutation is the product of its transpositions, so equals (−1)k(-1)^k(−1)k﻿, where kkk﻿ is the number of transposition in some decomposition, and since the det⁡\detdet﻿ is unique, so is the parity
